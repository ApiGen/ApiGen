    public static concat(string|string[]|self|null $name1, string|string[]|self|null $name2, array $attributes = []): static|null

    Concatenate two names, yielding a new Name instance. The type of the generated instance depends on which class this method is called on, for example Name\FullyQualified::concat() will yield a Name\FullyQualified instance. If one of the arguments is null, a new instance of the other name will be returned. If both arguments are null, null will be returned. As such, writing Name::concat($namespace, $shortName) where $namespace is a Name node or null will work as expected.
